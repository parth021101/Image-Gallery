## Project Overview

An interactive and visually appealing image gallery showcasing stunning photography from around the world. The gallery features a modern card-based layout with smooth hover animations that reveal image titles and favorite icons. Built with a clean, responsive design, it provides an engaging browsing experience for users exploring landscapes, urban scenes, and nature photography.

## Features

* **Responsive Grid Layout:** Flexbox-based card system that adapts to different screen sizes.
* **Hover Animation Effects:** Smooth slide-up gradient overlay appears on hover, revealing card details.
* **Image Cards:** Six beautifully styled cards with rounded corners and cover-fit images.
* **Interactive Elements:** Title display and heart icon (favorite marker) visible on hover.
* **Gradient Overlay:** Elegant black gradient from transparent to solid for better text readability.
* **Background Design:** Full-page background image with blur/gradient effect.
* **Modern UI:** Clean, minimalist design with white cards and smooth transitions.
* **Icon Integration:** Remix Icon library for consistent, high-quality icons.

## Technologies Used

* **HTML5:** For semantic markup and content structure.
* **CSS3:** Advanced styling with flexbox, gradients, transitions, and animations.
* **Google Fonts:** Poppins font family for modern typography.
* **Remix Icon:** Icon library for the heart/favorite icon
* **Unsplash API:** High-quality stock photography for card images and background
